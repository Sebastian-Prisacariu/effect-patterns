# Effect Patterns

A collection of practical patterns for building applications with [Effect-TS](https://effect.website).

## Patterns

- [Interface-First Development](./patterns/interface-first.md) — Build complete applications using only service contracts, then implement later

## Contributing

PRs welcome! Each pattern should include:
- Clear explanation of the problem it solves
- When to use it (and when not to)
- Complete, runnable code examples
- Real-world use cases
